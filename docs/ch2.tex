\section{Tensor}
Let $R$ be a commutative ring.
An $R$-tensor of shape $p$, is a map
\begin{align*}
    t : g(p) \to R
\end{align*}
A component of $t$ is denoted as
\begin{align*}
    t[i_1, \dots, i_r]
\end{align*}

The set of $R$-tensors of shape $p$ forms an $R$-linear space $\rtensor p$.
